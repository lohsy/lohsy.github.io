  • EGVD ImageJ Plugin

    Evolving Generalized Voronoi Diagram (EGVD) is a cell segmentation algorithm capable of segmenting touching cells. It requires 2 images to work: a cell image containing the cells and a seed image containing the cell nuclei. An existing program, StemCell3D, is already available and free for download. I created an EGVD plugin for ImageJ (a popular image processing software) to allow users to use this algorithm in ImageJ as well. You can download the jar file here. Put the jar file in ImageJ's plugin directory and you are good to go. Enjoy.
